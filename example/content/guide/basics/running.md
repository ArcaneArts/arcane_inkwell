## Development

```bash
cd your-project
jaspr serve
```

Visit http://localhost:8080

## Production Build

```bash
jaspr build
```

## With Base URL

For subdirectory hosting (e.g., GitHub Pages):

```bash
jaspr build --define=BASE_URL=/my-docs
```
